# Bodyfit — Brand Style Guide

Extracted from bodyfit.com.au's live theme (Elementor global styles). Update if the client supplies a formal brand book that differs.

---

## Color palette

| Role | Hex | Notes |
|---|---|---|
| **Primary** | `#FFFFFF` | White — dominant on hero backgrounds, text on dark surfaces |
| **Secondary** | `#010A11` | Near-black (subtle blue undertone) — page background, headlines, dark surfaces |
| **Accent** | `#FAD604` | Signature vivid yellow — used for CTAs, highlights, offer callouts |
| **Text** | `#FFFFFF` | Default body text on dark backgrounds |
| **Muted gray** | `#C3C3C3` (with 70% alpha → `#C3C3C3B2`) | Secondary text, captions |
| **Subtle overlay** | `#FFFFFF21` (white, 13% alpha) | Hero image overlays, divider accents |

**Core combo:** Black background + white text + yellow accent. High-contrast, bold, energetic — not pastel, not corporate.

### Use rules
- **Never** pair yellow (`#FAD604`) with a light or pastel background — it loses punch. Always on black/near-black.
- **CTA buttons:** yellow fill (`#FAD604`) with black text (`#010A11`), OR black fill with white text. No other combinations.
- **Headlines:** white on black, or black on white. Yellow reserved for emphasis words (single-word or short-phrase highlights) and CTAs.
- Do not introduce additional accent colors without approval.

## Typography

| Role | Font | Weight | Usage |
|---|---|---|---|
| **Display / Headlines** | **Bebas Neue** | 700 | All H1–H3, hero headlines, offer headlines. Always **UPPERCASE**, tight letter-spacing (-0.2px), tight line-height (0.85em–1em). Sizes 30px → 130px. |
| **Body** | **Gilroy** | 400 | Default body copy. 18px, line-height 1.3em |
| **Body — emphasis** | **Gilroy** | 600–700 | Sub-headings, button text, emphasis |
| **Fallback** | system sans-serif | — | If Bebas Neue / Gilroy unavailable |

### Type rules
- Display text (Bebas Neue) is **always uppercase** and **bold**. This is the core typographic signal of the brand.
- Never set Bebas Neue in lower-case or light weights — it loses brand recognition.
- Body copy (Gilroy) is sentence case. Do not uppercase body.
- Keep line-height tight on display (0.85–1em), generous on body (1.3em).
- For print/social creatives, substitute fonts with similar personality only if Bebas Neue + Gilroy aren't licensed (e.g. Oswald + Inter as secondary fallbacks — but request license first).

- **Nav/small use logo:** [bodyfit-logo-02-300x94.png](https://bodyfit.com.au/wp-content/uploads/2024/12/bodyfit-logo-02-300x94.png) — 300×94 PNG.
- **Style:** Lowercase wordmark, bold, minimalist. No symbol/icon mark.

### Logo rules
- Canonical written form in body copy: **Bodyfit** (capital B, lowercase rest). Logo itself displays lowercase.
- Maintain generous clear space around the logo — at least the height of the "b".
- Do not recolor the logo into yellow, red, or any non-brand color.
- Do not stretch, skew, or apply effects (shadow, gradient, outline).

## Buttons / CTAs

- **Primary CTA:** Yellow fill (`#FAD604`), black text (`#010A11`), Gilroy 700, uppercase, rounded corners (subtle).
- **Secondary CTA:** Transparent fill with white border + white text (on dark backgrounds).
- **CTA copy examples seen live:** "Join Now", "Book Trial", "Claim Offer Now". Short, action-led, first-person where appropriate.

## Photography & imagery rules

### Do
- Real gym footage — the actual Bodyfit floor, classes, equipment, locker rooms, fuel bars, sauna spaces
- Real members and real staff (with release)
- Diverse bodies — include adaptive members, LGBTQIA+ members, all age groups, all fitness levels
- Lifestyle over product — action shots, community moments, "day in the life"
- Persona-aligned imagery matching the 8 primary/growth/inclusivity personas (Emily, Josh, Jess, Matt, Michael, Nicole, Mo, Alex)
- Hero sections: high-quality lifestyle image or video with a semi-transparent dark gradient overlay for legibility
- Location-specific: each gym's own community, staff, and physical space

### Don't
- Don't use generic stock gym imagery
- Don't default to "ladies vs blokes" splits — audience is mixed and varies per location
- Don't feature only young, lean, conventionally fit bodies — contradicts the Inclusivity pillar
- Don't use intimidating gym-bro visuals (bulging veins, screaming faces, heavy-grunt "no pain no gain" aesthetics)

## Visual identity checklist

Before shipping any creative, confirm:

- [ ] Uses Bebas Neue for display + Gilroy for body
- [ ] Uses `#010A11` + `#FFFFFF` + `#FAD604` only (plus muted gray for body-secondary)
- [ ] Bebas Neue is UPPERCASE with tight letter-spacing
- [ ] CTA is yellow-fill with black text (or inverse)
- [ ] Photography is real Bodyfit members/staff/space — no stock
- [ ] Body/age/ability diversity represented
- [ ] Uses second person ("you"), not "our members"
- [ ] Avoids bro-speak, urgency theatre, corporate tone
